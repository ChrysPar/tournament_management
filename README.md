Καλώς ήρθατε στην εφαρμογή Tournament Management! 
Μέσω του σαιτ μπορορείτε να καταχωρείτε αθλητές και αυτοί να τοποθετούνται σε μια βάση δεδομένων. Όταν καταχωρηθούν όλοι οι αθλητές, τότε ξεκινάει η ζευγαροποίηση τους με τη βοήθεια του αλγορίθμου.
Ο αλγόριθμος ζευγαρώματος κοιτάει αρχικά ηλικία έπειτα κιλά και τέλος αγώνες που έχει παίξει ο κάθε αθλητής.
Όσοι αθλητές δεν ζευγαρωθούν εν τέλει μένουν διάθεσιμοι στην καρτέλα "Εκκρεμότητες". 
Τέλος, ο χρήστης μπορεί να κατεβάσει σε μορφή .xls τα ζευγάρια με κόκκινη και μπλε γωνία.
