💡 Πώς λειτουργεί η αποθήκευση στο Cloud
Αυτόματη Αποθήκευση: Κάθε αθλητής που καταχωρείται και κάθε αγώνας που δημιουργείται αποθηκεύεται άμεσα στο Cloud. Δεν χρειάζεται να κάνετε κάτι για να "σώσετε" την πρόοδό σας.
Αντίγραφα Ασφαλείας (Backups): Χρησιμοποιήστε τις επιλογές Εξαγωγή στις καρτέλες Αθλητών και Αγώνων για να κρατάτε αρχεία στον υπολογιστή σας.
Συνέχεια Εργασίας: Απλά ανοίξτε το link από οποιονδήποτε υπολογιστή. Τα δεδομένα θα φορτωθούν αυτόματα από τη βάση δεδομένων.
Εισαγωγή από Αρχεία: Αν έχετε ήδη λίστες σε Excel, μετατρέψτε τις σε CSV και χρησιμοποιήστε τα παραπάνω πεδία για να τις ανεβάσετε μαζικά στο Cloud.
